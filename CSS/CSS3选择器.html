CSS3选择器

1.属性(html属性)选择器

  • 语法:html标签[html标签属性]{css样式}

    说明:用于选取带有指定属性的元素.

    举例: 

    <style>
img[alt] {
color: red
}
</style>
<img src="" alt="test1">
<img src="" alt="test2">

  • 语法:html标签[html标签属性="值"]{css样式}

    说明:用于选取带有指定属性和值的元素.

    举例: 
    <style>
input[type="submit"] {
width: 500px;
height: 500px
}
</style>

<div>
<input type="text" name="" value="">
<input type="submit" name="" value="">
</div>
